# Student Budget Analysis Project

## Overview

This project analyzes a student's personal budget and spending behavior using Excel and SQL.

The objective was to improve data analysis skills by solving real-world business questions using a structured dataset.

## Tools Used

* Microsoft Excel
* MySQL
* Data Cleaning
* Pivot Tables
* Charts
* SQL Aggregations
* Subqueries
* Window Functions

## Project Workflow

### Excel Analysis

* Solved 30 business-oriented questions
* Used SUMIFS, COUNTIFS, IF, INDEX-MATCH, VLOOKUP, Pivot Tables, and Charts
* Created category-wise spending analysis
* Identified spending trends and patterns

### SQL Analysis

Solved 30 SQL questions ranging from beginner to intermediate level.

Topics covered:

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* CASE WHEN
* Aggregate Functions
* Subqueries
* ROW_NUMBER()
* DENSE_RANK()
* LAG()
* PERCENT_RANK()
* Running Totals
* Rolling Averages

## Key Insights

* Identified major spending categories
* Compared Need vs Want expenses
* Analyzed monthly spending trends
* Ranked transactions within categories
* Calculated rolling averages and spending concentration

## Learning Outcomes

This project helped strengthen practical skills in:

* Excel Data Analysis
* SQL Query Writing
* Data Cleaning
* Business Problem Solving
* Data Storytelling
